# Слова.Words — тренажёр английского

Локальный тренажёр для запоминания английских слов, форм глаголов и выражений.
Интервальное повторение (как в Anki), автосохранение прогресса, работа офлайн.
В базе ~780 карточек: слова, неправильные глаголы и выражения (фразовые глаголы, идиомы, связки).

## Запуск

- **Windows:** двойной клик по **`run.bat`**.
- **Linux / macOS:** `chmod +x run.sh` (один раз), затем `./run.sh`.

Скрипт сам создаст окружение, поставит зависимости и поднимет сервер.
После запуска открыть в браузере: **http://127.0.0.1:8000**

Требуется **Python 3.10+** (на Windows при установке отметить «Add Python to PATH»).
Версию без Python можно собрать одним `.exe` — см. ниже.

**Остановить:** вкладка «Ещё» → «Завершить работу», либо закрыть окно консоли,
либо запустить `stop.bat` (Windows).

## Как пользоваться

- Вкладки: **Учить** (слова), **Формы** (глаголы V1/V2/V3 и исключения),
  **Выражения** (фразовые глаголы, идиомы, связки), **Добавить**, **Статистика**, **Ещё**.
- Показывается карточка → вспоминаете ответ → **«Знал»** / **«Не знал»**.
  Что выучено — показывается реже, ошибочное — чаще.
- Клавиатура: `Пробел`/`Enter` — показать; `→` или `2` — Знал; `←` или `1` — Не знал.
- Прогресс сохраняется автоматически. Бэкап и обмен словами — во вкладке «Ещё».

Данные хранятся только локально (папка `data/` рядом с приложением); ничего не отправляется в сеть,
кроме автоподстановки транскрипции при добавлении новых слов.

## Сборка автономного .exe (Windows)

Запустить **`build.bat`** — получится `dist\vocab-trainer.exe` (~15 МБ).
Этому файлу не нужен установленный Python: двойной клик → http://127.0.0.1:8000.
Прогресс пишется в папку `data\` рядом с `.exe`.

> **Про антивирус.** `.exe`, собранный упаковщиком (PyInstaller), часть антивирусов и
> SmartScreen могут пометить как подозрительный и удалить — это **ложное срабатывание**,
> типичное для «запакованного в один файл» Python. Файл безопасен (исходники — рядом, в `app.py`
> и `static/index.html`). Что можно сделать: добавить файл в исключения антивируса, либо
> не пользоваться `.exe` и запускать через `run.bat` с установленным Python, либо собрать `.exe`
> самостоятельно командой `build.bat`.

## Что внутри (для разработчиков)

- `static/index.html` — всё приложение (интерфейс, логика, встроенный словарь). Зависимостей нет.
- `app.py` — локальный сервер на FastAPI: отдаёт страницу и хранит прогресс в `data/state.json`.
  При открытии `index.html` напрямую (без сервера) работает на хранилище браузера.
- `requirements.txt` — зависимости для запуска через Python (fastapi, uvicorn).

Порт по умолчанию `8000`, можно переопределить переменной окружения `PORT`.

## Лицензия / приватность

Приложение не собирает и не передаёт персональные данные — весь прогресс остаётся на устройстве.
